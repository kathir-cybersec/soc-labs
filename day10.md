# Day 10 – Common Cyber Threats

 What I Learned
- Cyber attacks are methods used to compromise systems
- Common attacks include phishing, malware, brute force, and DDoS

 Attack Types

 Phishing
- Fake emails used to steal information

 Malware
- Malicious software that harms systems

 Brute Force
- Repeated login attempts to guess passwords

 DDoS
- Overloading a system with traffic

 SOC Perspective
- SOC analysts detect attacks using logs and network data
- Each attack has specific indicators

 My Thoughts
- Understanding attacks helps in detecting them
- I need to learn how to identify these in real systems
