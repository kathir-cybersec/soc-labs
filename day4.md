 Day 4 – Linux File Permissions

 What I Learned
- Linux controls access using permissions
- Permissions include read (r), write (w), execute (x)
- There are 3 levels: user, group, others

 Commands I Learned
- ls -l: view permissions
- chmod: change permissions
- chown: change file owner

 Hands-On Practice
- Completed Bandit Level 4–7 on OverTheWire

 Example Understanding
- rwxr-xr-- means:
  - Owner: full access
  - Group: read + execute
  - Others: read only

 My Thoughts
- Permissions are important for security
- Misconfigured permissions can cause vulnerabilities
