 Day 9 – Network Traffic Analysis (Wireshark Basics)

 What I Learned
- Wireshark is used to capture and analyze network traffic
- It shows packets moving between devices
- Each packet contains source, destination, protocol, and port

 Important Concepts
- Packets carry data across networks
- TCP and UDP are communication protocols
- Ports identify services like HTTP (80) and HTTPS (443)

 SOC Perspective
- Analysts monitor traffic to detect suspicious activity
- Unknown IPs or unusual ports may indicate threats

 Hands-On Practice
- Learned packet structure and traffic analysis basics

 My Thoughts
- Network analysis is interesting and important
- I want to learn more about detecting attacks using traffic
