 Day 5 – Processes and System Monitoring

 What I Learned
- A process is a running program in the system
- Multiple processes run at the same time
- Malware can also run as a hidden process

 Commands I Learned
- ps: show running processes
- top: monitor processes in real time
- kill: stop a process
- whoami: shows current user
- history: shows command history

 Hands-On Practice
- Completed Bandit Level 8–10
 Why This is Important for SOC
SOC analysts monitor systems to detect suspicious processes.
