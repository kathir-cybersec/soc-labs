Day 8 – Network Traffic Basics

 What I Learned
- Network traffic is data moving between devices
- Data is sent in packets
- Each packet contains source and destination information

Important Concepts
- IP Address: identifies devices
- Port: identifies service (e.g., 443 for HTTPS)
- Protocol: rules for communication (TCP/UDP)

 SOC Perspective
- Attackers generate unusual network traffic
- Monitoring traffic helps detect threats
