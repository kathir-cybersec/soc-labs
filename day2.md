Day 2 – Networking Basics

 What I Learned
- The internet connects devices worldwide
- When we open a website, a request is sent to a server
- DNS converts domain names into IP addresses

 Important Concepts
- TCP: Reliable communication protocol
- UDP: Faster but less reliable
- Ports help identify services:
  - 80 → HTTP
  - 443 → HTTPS
  - 22 → SSH
